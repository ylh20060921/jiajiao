# 家教管理系统

一个基于浏览器的家教管理系统，支持教师、家长、管理员三种角色。

## 功能特点

- **教师端**：课程管理、学生管理、排课、订单、收入统计
- **家长端**：查看课程、报名、支付、个人信息
- **管理员端**：教师审核、课程管理、系统设置

## 部署说明

本项目为单页应用，可直接部署到 GitHub Pages、Vercel 等静态托管服务。

### 本地开发
直接打开 `index.html` 即可运行。

### 在线部署
1. 上传到 GitHub 仓库
2. 使用 Vercel 或 GitHub Pages 部署

## 技术栈
- 纯 HTML/CSS/JavaScript
- 无依赖，无需构建
- 响应式设计，支持移动端
